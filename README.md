# Discovery Lab 科研工作台

Discovery Lab 的静态科研门户，可直接部署到 GitHub Pages。首页集中展示实验室常用工具，并内置免疫荧光强度分析模块。

## 免疫荧光分析模块

打开 `immunofluorescence.html`，上传 PNG、JPEG 或 WebP 图像即可在浏览器本地完成：

- 自动识别红、绿、蓝荧光通道，也可手动指定；
- 按指定分位数估计并扣除均匀背景；
- 使用 Otsu 自动阈值或手动阈值分割阳性信号；
- 过滤小面积噪点，生成可检查的叠加图和二值蒙版；
- 计算背景扣除后的平均荧光强度、积分密度、阳性面积占比和信噪比；
- 导出 CSV、PNG 结果图和中英文方法描述。

图像只在当前浏览器内处理，不会上传到服务器。

## 本地预览

这是一个无构建步骤的静态网站。可直接打开 `index.html`，或使用任意静态文件服务器预览。

## GitHub Pages 部署

在仓库的 **Settings → Pages** 中，将来源设置为 **Deploy from a branch**，选择 `main` 分支和根目录 `/`。保存后即可发布。

## 定量注意事项

- 组间图像必须使用相同曝光、增益、物镜和采集流程。
- 优先使用无压缩 PNG；JPEG 压缩会改变像素强度。
- 当前模块基于浏览器解码后的 8-bit 全图数据。16-bit TIFF、复杂 ROI、细胞级分割或共定位分析建议使用 Fiji/ImageJ 等经过验证的流程。
- 发表前应目视检查分割蒙版，并在方法中报告通道、背景分位数、阈值方法和最小区域参数。
